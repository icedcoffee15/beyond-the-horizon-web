Playing With a Friend

Beyond The Horizon supports 2-player co-op online via Radmin VPN.

01

Install Radmin VPN

Both players download and install Radmin VPN (free) from radmin-vpn.com. It creates a virtual LAN so you connect as if on the same network.

02

Create or Join a Network

One player clicks Create Network with a name and password. The other clicks Join Network and enters those same credentials.

03

Host the Session

The host opens the game, goes to Multiplayer → Host Game. Your Radmin VPN IP (e.g. 26.x.x.x) is shown in the lobby. Share it with your friend.

04

Join and Play

The second player goes to Multiplayer → Join Game, enters the host's Radmin VPN IP, and clicks Connect. Both players spawn on the map.

Both players must be on the same Radmin VPN network. Standard LAN play is also supported if you're on the same WiFi.

🤝 Play With a Friend

Beyond The Horizon supports 2-player co-op. Here's how to connect with a friend online.

1

Install Radmin VPN

Both players download and install Radmin VPN (free) from radmin-vpn.com. This creates a virtual private network so you can play like you're on the same WiFi.

2

Create or Join a Network

One player clicks Create Network and sets a name + password. The other player clicks Join Network and enters those same credentials.

3

Host the Game

The host opens the game → Main Menu → Multiplayer → Host Game. The Radmin VPN IP is shown (e.g. 26.x.x.x). Share it with your friend.

4

Join the Game

The second player opens the game → Main Menu → Multiplayer → Join Game → enters the host's Radmin VPN IP → click Connect. Both players spawn on the map — game begins!

⚠️ Both players must be on the same Radmin VPN network for this to work. Standard LAN play also works if you're on the same WiFi.
